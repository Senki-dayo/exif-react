## exif情報を扱うReactライブラリ

### exif-react
exif-jsを利用してexif情報を取得するReactコンポーネント  
### exifr-react
exifr-jsを利用してexif情報を取得するReactコンポーネント

### how to start
`npm install`
`npm start`
